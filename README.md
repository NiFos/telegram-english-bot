## Tech stack

* [Telegraf.js](telegraf.js.org/): Telegram bot framework.
* [MongoDB](https://www.mongodb.com/): NoSql database

## Features

* List learned / to learn words

* Delete account 

## How to start locally

```
1) Clone repo
2) yarn install
3) Create and edit .env file how in .env.example 
4) yarn dev
```
